# Workers Program

A Python program used to manage workers information including **name**, **ID**, **price**, and **quantity**.  
This project is a simple console-based application that demonstrates basic Python programming concepts.

## Features

- Add worker details (name, ID, price, quantity)
- View all workers information
- Update worker data
- Delete worker records
- Easy to use command line interface

## Technologies Used

- **Python**
